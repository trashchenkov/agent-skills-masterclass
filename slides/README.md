# Slidev-презентация: теория Agent Skills

Файл презентации: `slides.md`.

На каждом слайде есть зарезервированное место под QR-код в правом нижнем углу.
QR предполагается заменить ссылкой на репозиторий с материалами мастер-класса.

## Запуск

Из этой папки:

```bash
pnpm install
pnpm dev
```

## Экспорт в PDF

```bash
pnpm export
```

## Где заменить ссылку на репозиторий

В последних слайдах замените плейсхолдеры:

- `REPO_URL` — URL репозитория;
- `REPO_DIR` — имя папки после `git clone` или распаковки архива.

Если нужен реальный QR вместо плейсхолдера, замените компонент
`global-bottom.vue`: вставьте изображение QR-кода вместо блока `.qr-box`.
